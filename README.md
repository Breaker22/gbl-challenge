## Ejecucion de proyecto
```
1) Descargar el proyecto con git clone
2) Instalar gradle y java 8
3) Abrir una consola cmd y ejecutar lo siguiente:

gradle build
./gradlew bootRun
```

## Dependencias
* Spring 2.5.14
* Swagger
* Actuator
* H2
